# 键盘快捷键

绝大多数编辑器快捷键都可在 **设置 > 快捷键** 中自定义。以下默认值与全新安装时一致。

## 画布操作

这些是手势型操作，不支持自定义绑定。

| 快捷键 | 动作 |
| --- | --- |
| `Ctrl` + 鼠标滚轮 | 缩放 |
| `Ctrl` + 拖动 | 平移画布 |
| 触控板捏合 | 捏合缩放 |

## 工具切换

| 默认按键 | 动作 |
| --- | --- |
| `V` | 切换到选择工具 |
| `M` | 切换到块工具（创建文本块） |
| `B` | 切换到笔刷工具 |
| `E` | 切换到橡皮擦工具 |
| `R` | 切换到修复笔刷工具 |

## 笔刷大小

| 默认按键 | 动作 |
| --- | --- |
| `]` | 增大笔刷大小（上限 128） |
| `[` | 减小笔刷大小（下限 8） |

## 历史与选择

| 默认按键 | 动作 |
| --- | --- |
| `Ctrl` + `Z`（macOS 上为 `Cmd` + `Z`） | 撤销 |
| `Ctrl` + `Shift` + `Z`（macOS 上为 `Cmd` + `Shift` + `Z`） | 重做 |
| `Ctrl` + `Y` | 重做（旧版备用，不可重新绑定） |
| `Ctrl` + `A`（macOS 上为 `Cmd` + `A`） | 选中当前页面所有文本块 |

撤销和重做即使在文本框内输入时也会刻意触发——场景历史的优先级高于浏览器原生的文本撤销。`Ctrl + A` 只在文本框外触发，因此 textarea 与 input 内仍保留原生的“全选文本”行为。

## 自定义快捷键

打开 **设置 > 快捷键** 即可重新绑定上面任何可自定义的快捷键。冲突会被高亮显示，你也可以在同一界面恢复默认值。

工具切换与笔刷大小快捷键只有在键盘焦点不在可编辑文本框时才会触发，所以不会打断你在 OCR 或翻译面板中的输入。
